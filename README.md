
# Attendance Tracker

## Description
A simple web app to track daily attendance per subject and calculate percentage.

## Features
- Add subjects
- Mark attendance
- Auto percentage calculation

## Tech Stack
- Python (Flask)
- HTML, CSS

## How to Run
1. Install Flask
2. Run app.py
3. Open localhost
